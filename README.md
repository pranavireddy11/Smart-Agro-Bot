# 🌾 Smart Agro Bot

An AI-powered farming assistant designed to help Indian farmers with crop guidance, soil recommendations, irrigation advice, fertilizer suggestions, and multilingual voice interaction.

## Features

- 🌱 Crop and soil recommendations
- 💧 Irrigation guidance
- 🌦 Weather-related information
- 🐛 Pest management suggestions
- 🎤 Voice input support
- 🔊 Text-to-speech responses
- 🌐 Multilingual support
  - English
  - Hindi
  - Telugu
- 📡 Offline/SMS mode simulation
- 📷 Crop and soil image upload preview

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Web Speech API
- Google Translate API
